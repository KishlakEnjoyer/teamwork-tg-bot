# AI Personality Analyzer & Music Recommender Telegram Bot

## Project Description

This project is an intelligent Telegram bot that analyzes the user's emotional state and personality based on their messages and provides personalized music recommendations using AI and external music APIs.

The bot uses modern Natural Language Processing (NLP) techniques and transformer-based models to detect emotions such as joy, sadness, anger, fear, and neutrality. Based on this analysis, the system generates a personality profile and recommends music tracks that match the user's emotional state.

The system is built using a scalable backend architecture with FastAPI, PostgreSQL database, and integration with the Spotify Web API.

---

## Key Features

* Emotion detection using transformer-based AI models
* Personality profile generation based on message analysis
* Personalized music recommendations via Spotify API
* Telegram bot interface for real-time interaction
* REST API backend using FastAPI
* Persistent data storage using PostgreSQL
* Modular and scalable architecture

## How It Works

1. User sends a message to the Telegram bot
2. The bot sends the message to the FastAPI backend
3. The AI model analyzes the emotional content of the message
4. The system updates the user's personality profile
5. The bot requests music recommendations from Spotify API
6. The bot sends personalized recommendations back to the user
